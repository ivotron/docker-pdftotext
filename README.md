# docker-pdftotext

### Usage
```
# Convert from in.pdf to out.txt
docker run --rm -i kalledk/pdftotext < in.pdf > out.txt

# Convert from in.pdf to stdout
docker run --rm -i kalledk/pdftotext < in.pdf
```
